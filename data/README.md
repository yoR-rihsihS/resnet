Download the dataset from https://www.kaggle.com/c/ifood-2019-fgvc6/overview and extract here.

Make sure the directory structure is as follows:
```
data/
├── test_set/
│   ├── test_000000.jpg
│   ├── test_000001.jpg
│   └── ...
├── train_set/
│   ├── train_000000.jpg
│   ├── train_000001.jpg
│   └── ...
├── val_set/
│   ├── val_000000.jpg
│   ├── val_000001.jpg
│   └── ...
├── class_list.txt
├── ifood2019_sample_submission.csv
├── sample_submission.csv
├── train_labels.csv
└── val_labels.csv
```
